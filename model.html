Model¶

Models are used to create neural networks in SimFlow

class model.Model[source]¶

Represents a neural network with any combination of layers

add_layer(layer)[source]¶

Adds a layer to the network in a sequential manner. The input to this layer will be the output of the last added layer or the initial inputs to the networks if this is the first layer added.

Args:
layer (Layer):Layer to be added to the model
