data_loader_mnist module¶

data_loader_mnist.load_normalized_mnist_data_conv()[source]¶
Loads and normalizes the MNIST data. Reads the data from
data/mnist_train.csv data/mnist_test.csv

If these are not present they are downloaded and saved from https://pjreddie.com/projects/mnist-in-csv/

Returns two dictionaries, input and labels Each has keys ‘train’, ‘val’, ‘test’ which map to numpy arrays
