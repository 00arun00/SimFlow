Layers¶

Description of Class Layers

All SimFlow layers have the following methods

  • layer.forward(X): return the output after passing it throught the layer
  • layer.backward(dY): returns the gradient after backpropogating it throught the layer
  • layer.get_params( ): returns the params of the layer as a list of Numpy arrays.
  • layer.set_params(new_params): sets the weights of the layer from a list of Numpy arrays (with the same shapes as the output of get_params)

class layers.BN(dim, *, elr=0.9, trainable=True)[source]¶

Batch normalization Layer (Full)

During Train:
BN(x) = gamma(X - mean(X_batch))/std(X_batch) + beta
During Test:
BN(x) = gamma((X - mean_learned)/Learned_std) + beta
Initialization:
beta:initialized to zeros
gamma:initialized to zeros
mean_learned:initialized to zeros
var_learned:initialized to zeros
Args:
dim (int):

size of input passed
elr (float):

exponential learning rate for updating mean_learned
trainable (bool):
 
False:parameters of the layer are frozed
True:parameters are updated during optimizer step

class layers.Dense(input_dim, output_dim, *, init_method='Xavier', trainable=True)[source]¶

Dense / Linear Layer

Represent a linear transformation Y = X*W + b
X:is an numpy.ndarray with shape (batch_size, input_dim)
W:is a trainable matrix with dimensions (input_dim, output_dim)
b:is a bias with dimensions (1, output_dim)
Y:is an numpy.ndarray with shape (batch_size, output_dim)
Initialization:
W:initialized with either Xavier or He initialization
b:initialized to zero

size of output requred
