class layers.BN(dim, *, elr=0.9, trainable=True)[source]¶

Bases: layers.Layer

Represents a Batch normalization Layer (BN)

During Train : BN(x) = gamma(X - mean(X_batch))/std(X_batch) + beta

During Test : BN(x) = gamma((X - Learned_mean)/Learned_std) + beta

class layers.Dense(input_dim, output_dim, *, init_method='Xavier', trainable=True)[source]¶

Bases: layers.Layer

Dense / Linear Layer Represent a linear transformation Y = X*W + b

X is an numpy.ndarray with shape (batch_size, input_dim) W is a trainable matrix with dimensions (input_dim, output_dim) b is a bias with dimensions (1, output_dim) Y is an numpy.ndarray with shape (batch_size, output_dim)

Returns:
dX (numpy.ndarray) : Input gradient after backpropagating dY through Dense layer var_grad_list (list): If Layer is trainable contains (W,dW) and (b,db) otherwise []
